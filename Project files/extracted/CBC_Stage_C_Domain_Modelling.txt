1. Introduction
This document presents the Stage C Domain Modelling output for the CBC Digital Learning
Platform. It builds directly on the functional requirements in the SRS (v1.0) and the use cases
locked in Stage B to identify and formally describe the core domain entities of the system — the
real-world concepts the platform must represent and manage.
Domain modelling at this stage captures what entities exist, what they know (attributes), and how
they relate to each other (associations, compositions, and inheritance). This is distinct from
database schema design (Stage F), which will derive from this model.
